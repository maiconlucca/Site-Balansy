Site oficial do app Balansy, hospedado no GitHub Pages.

## Estrutura

```
balansy-site/
├── index.html          ← Página principal (tudo em um arquivo)
└── assets/

    ├── screen-tasks.png

    ├── screen-habits.png
    ├── screen-habits2.png

    └── screen-habits-index.png
```

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `balansy-site`)
2. Faça upload de todos os arquivos mantendo a estrutura de pastas
3. Vá em **Settings → Pages**
4. Em **Source**, selecione `main` branch e pasta `/root`
5. Salve. O site estará disponível em `https://seuusuario.github.io/balansy-site`

## Idiomas suportados

- 🇧🇷 Português (padrão para pt-BR e pt-PT)
- 🇺🇸 Inglês
- 🇪🇸 Espanhol
- 🇮🇹 Italiano

O idioma é detectado automaticamente pelo sistema. O usuário pode trocar manualmente no canto superior direito.
